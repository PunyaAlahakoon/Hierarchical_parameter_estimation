This folder contain the datasets and other known factors that are used in the simulation experiment in the paper 

sc2_true_betas.mat==== vector containing the true $\beta$ values of the sub-populations in Population B. 


sc1_data.mat====== synthetic data of Population A. each column shows the daily prevalnace data of each sub-population 


sc2_data.mat====== synthetic data of Population B. each column shows the daily prevalnace data of each sub-population 


sc6_true_betas.mat==== vector containing the true $\beta$ values of the sub-populations in Population C. first 15 elements are true parameter values of Population B. 

sc6_data.mat==== synthetic data of Population C. first 15 columns are synthetic data from Population B.
